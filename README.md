# Blockchain Concepts Simulator

This repository explains core blockchain concepts in a simple and beginner-friendly way.

## Topics Covered
- Blockchain Basics
- Hashing Algorithm
- Immutable Ledger
- Byzantine Generals Problem
- Consensus Protocols (PoW, PoS)
- Forks and 51% Attack

## Objective
To demonstrate strong conceptual understanding of blockchain fundamentals without coding.
